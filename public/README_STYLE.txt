
README_STYLE.txt - Utilisation du style CSS standard BuzzSound

Déposez ce fichier sous :
/public/css/style.css

Ce style est le thème visuel principal du jeu. Il s'applique à toutes les pages :
- accueil
- menu
- quiz
- résultats
- crédits

Il comprend :
✔ Typographie claire
✔ Boutons arrondis cohérents
✔ Couleurs harmonisées (bleu BuzzSound + jaune Buzz)
✔ Animation pulsante pour le bouton Buzzer
✔ Mise en page responsive

Prochaine étape :
Inclure <link rel="stylesheet" href="{{ asset('css/style.css') }}"> dans le layout blade de base.
